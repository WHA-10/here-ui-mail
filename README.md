# 📧 Mail

یک پروژه آزمایشی رایانامه (ایمیل) با رابط کاربری تستی.

این مخزن برای آزمایش ایده‌ها، ساخت رابط کاربری ساده و تست مفاهیم مربوط به سیستم‌های رایانامه ایجاد شده است.

---

## ✨ معرفی

**Mail** یک پروژه تستی برای شبیه‌سازی یک محیط ساده رایانامه است.  
هدف این پروژه بیشتر یادگیری، آزمایش رابط کاربری و بررسی ساختار یک سیستم ایمیل است.

---

## 🚀 ویژگی‌ها

- 📬 رابط کاربری ساده برای تست
- 🧪 پروژه آزمایشی برای ایده‌های مربوط به ایمیل
- 🧩 ساختار ساده برای یادگیری و توسعه
- 🔧 مناسب برای آزمایش و توسعه اولیه

---

## 🎯 هدف پروژه

این پروژه یک **پروژه تستی** است و برای استفاده در محیط واقعی طراحی نشده است.  
هدف آن بیشتر آزمایش ایده‌ها و تمرین ساخت یک سیستم رایانامه ساده است.
